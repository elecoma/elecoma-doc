配送設定
========

配達業者の登録を行います


配送業者・配送料を追加 ボタンをクリックすると :ref:`admin_shops_basic_delivery_setting` へ移行します

.. list-table::
   :widths: 50 400

   * - 配送業者
     - 実際に配送業者選択時に表示される名称です
   * - 編集
     - :ref:`admin_shops_basic_delivery_setting` 画面へ移行し選択した配送業者の設定を編集します
   * - 削除
     - 選択した配送業者を配送業者リストから末梢します
   * - 移動
     -
